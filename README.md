# Yew Numbers API
Simple frontend for the [Numbers API](http://numbersapi.com/) written in Rust using Yew.rs


## Development


### Prerequisites

- Rust
- Node.js
- Trunk


### Getting Started

Clone the repository
```
git clone https://github.com/SushiWaUmai/yew-numbers-api.git
cd yew-numbers-api
```


Start developing
```
trunk serve --open
```
